174 """
175 cgrad(colors, [values]; categorical = nothing, scale = nothing, rev = false, alpha = nothing)
176
177 Construct a Colorgradient from `colors` and `values`.
178
179 `colors` can be a symbol for ColorSchemes.jl `ColorScheme`s, a `ColorScheme`, a vector of colors, a `ColorGradient` or a `ColorPalette`.
180 If `values` is an integer, it specifies the numbers of colors chosen equidistantly from the colorscheme specified by colors.
181 Otherwise vectors are accepted.
182 For continuous color gradients `values` indicate where between 0 and 1 the colors are positioned.
183 For categorical color gradients `values` indicate where a color ends and where a new one begins between 0 and 1.
184 0 and 1 are added to `values` if not already present.
185
186 If `rev` is `true` colors are reversed.
187 `scale` accepts the symbols `:log`, `:log10`, `:log2`, `:ln`, `:exp`, `:exp10` or functions.
188 If `alpha` is set, it is applied to all colors.
189 """

195 rev = false,
196 alpha = nothing,

201
202 if alpha ≢ nothing
203 rgbs = convert.(RGB, colors.colors)
204 colors = ColorScheme(RGBA.(rgbs, alpha))
205 end
206 rev && (colors = reverse(colors))
207 values = if scale ∈ (:log, :log10) || scale ≡ log10
208 log10.(ColorSchemes.remap(values, 0, 1, 1, 10))
